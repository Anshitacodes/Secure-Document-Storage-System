# 🔐 Secure Document Storage System

This is a web-based application developed using ASP.NET and C#.

## 📌 Features
- Secure Login & Registration
- Upload Documents
- View Uploaded Documents
- Delete Documents
- Dashboard showing total document count
- SQL Server Database Integration

## 🛠 Technologies Used
- ASP.NET
- C#
- SQL Server
- HTML
- CSS

## 🎯 Project Objective
To build a secure document management system that allows users to safely store and manage their documents online.
